# Campus Notifications Frontend Submission

This repository is organized for the frontend track.

## Structure

- `notification_app_fe/` - runnable React/Vite frontend application.
- `notification_app_be/` - frontend-track placeholder; no backend is required.
- `logging_middleware/` - reusable frontend logging middleware.
- `notification_system_design.md` - Stage 1 priority inbox approach and design notes.
- `ScreenShorts/` - screenshots for submission.

## Run the Frontend

```bash
cd notification_app_fe
npm install
npm run dev
```

The application runs at:

```text
http://localhost:3000
```
